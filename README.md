# ru-address-normalizer
Проект созданный для решения задачи нормализации русских адресов через DL

Префиксы классов:
- `B-`: Начало именованной сущности (такие слова как страна, город и тп)
- `I-`: Продолжение именованной сущности (определенный город, например Крымск)
- `O`: Токен не относится к именованной сущности

Основные классы:
- `Country` - Страна
- `Republic` - Республика
- `Krai` - Край
- `Oblast` - Область
- `AO` - Автономный округ
- `District` - Район
- `City` - Город
- `Selo` - Село
- `Village` - Деревня
- `Poselok` - Поселок
- `Municipality` - Муниципальное образование
- `Street` - Улица
- `Avenue` - Проспект
- `Boulevard` - Бульвар
- `Lane` - Переулок
- `Highway` - Шоссе
- `Embankment` - Набережная
- `Microdistrict` - Микрорайон
- `Quarter` - Квартал
- `House` - Дом
- `Building` - Строение
- `Structure` - Сооружение
- `Flat` - Квартира
